# Data Structures and Algorithms Lab Repository

This repository contains the lab experiments and practice programs completed as part of the Data Structures and Algorithms (DSA) Lab coursework.

## Topics Covered

The labs include concepts such as:

- Arrays
- Linked Lists
- Stacks
- Queues
- Trees
- Binary Search Trees
- Graphs
- Searching Algorithms
- Sorting Algorithms
- Recursion
- Hashing
- Dynamic Programming

## Languages Used

- C
- C++

## Purpose

This repository is maintained for academic learning, practice, and reference purposes.
